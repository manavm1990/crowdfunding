# User Authentication and Verification Demo

- ESM
- Linting, Prettier
- MySQL, Sequelize
- Node, Express
- `nodemon` (`npm start`)

You will need to provide the following in your `.env`:

```bash
DB_NAME=user_db
DB_USER
DB_PASSWORD
PORT
EMAIL
```

You will need to `SOURCE app/db/schema.sql` to create `user_db`.
